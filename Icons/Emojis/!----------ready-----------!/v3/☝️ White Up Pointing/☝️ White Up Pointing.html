☝️ Index Pointing Up

Emoji Meaning

A single finger pointing upward, which could be used to represent the number one, or to ask a question.

Index Pointing Up was approved as part of Unicode 1.1 in 1993 under the name “White Up Pointing Index” and added to Emoji 1.0 in 2015.
